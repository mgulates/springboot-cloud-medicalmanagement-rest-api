# Spring Boot Medical Management REST API

This project is a simple Java Spring Boot REST API application developed as part of the SWE212 course. It provides basic CRUD operations for managing medical entities such as patients, doctors, and appointments.

## Technologies
- Java 25
- Spring Boot 4.0.1
- Maven
- MySQL / PostgreSQL
- Nginx (Reverse Proxy)
- AWS / Azure Cloud

## Features
- RESTful API design
- CRUD operations (Create, Read, Update, Delete)
- Layered architecture (Controller, Service, Repository)
- Database integration

## Deployment
The application is deployed on a Linux-based cloud instance using Nginx as a reverse proxy.

## Testing
All endpoints are tested locally and on the cloud using Postman.
